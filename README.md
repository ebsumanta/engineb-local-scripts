## How to use this Repo

```
groupper.py
___________________
This is grouper databricks code which supports local execution with pandas. It required `grouper` directory data handling.

change is_local=False to run on Databricks
change is_local=true to run in local


```

```
cdm_notebook.py
___________________
This is CDM Mapped Calculated notebook 
This requires container directory and some configuration files in order to run

set local = False for databricks execution
set local = True for local execution
```
